CSS Bonus — Référentiel complet

Objectif

Fournir un inventaire complet des propriétés CSS et bonnes pratiques pour construire des pages modernes et accessibles.

Propriétés CSS essentielles

  • Coloration : color, background-color
  • Textes : font-family, font-size, font-weight, text-align
  • Box model : width, height, margin, padding, border
  • Flexbox : display: flex, justify-content, align-items, flex-wrap
  • Grid : display: grid, grid-template-columns, gap
  • Animations et transitions : transition, @keyframes, animation
  • Positionnement : position, top, left, z-index
  • Pseudo-classes et pseudo-éléments : :hover, :focus, ::before, ::after

Bonnes pratiques

  • Organiser le CSS en sections logiques (typographie, layout, couleurs, composants).
  • Utiliser des classes plutôt que des ids pour les styles réutilisables.
  • Préférer des unités relatives (em, rem, %) pour le responsive.
  • Minimiser les répétitions en utilisant des classes et variables CSS (CSS custom properties).
  • Valider le CSS et tester sur différents navigateurs et écrans.
