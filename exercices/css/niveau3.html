CSS — Niveau 3 : Box model et marges

Objectif

Comprendre le box model CSS, la taille des éléments, les marges, padding et bordures.

Exemple de code

div {
  width: 200px;
  height: 100px;
  margin: 20px;
  padding: 10px;
  border: 2px solid black;
  background-color: lightgrey;
}
  • width/height : dimensions de l’élément.
  • margin : espace extérieur à l’élément.
  • padding : espace intérieur entre le contenu et la bordure.
  • border : bordure autour de l’élément.
  • background-color : couleur de fond.

Exercices

  1. Créer un div avec largeur et hauteur fixées.
  2. Ajouter un padding et une marge différents.
  3. Appliquer une bordure visible.
  4. Observer comment les dimensions totales changent avec padding et border.

Astuces et bonnes pratiques

  • Se rappeler que width/height ne comprennent pas padding et border par défaut.
  • Utiliser box-sizing: border-box; pour inclure padding et border dans les dimensions.
  • Tester avec différents padding/margin pour comprendre l’espacement.
