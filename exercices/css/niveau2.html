CSS — Niveau 2 : Couleurs, polices et fonds

Objectif

Apprendre à modifier les couleurs, les polices et les fonds d’une page web pour améliorer le design.

Exemple de code

body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
  color: #333333;
}

h1 {
  color: darkblue;
}

p.important {
  color: red;
  font-weight: bold;
}
  • background-color : couleur de fond de l’élément.
  • font-family : type de police.
  • color : couleur du texte.
  • font-weight : épaisseur du texte.

Exercices

  1. Appliquer un fond clair à la page.
  2. Changer la couleur et la police du titre principal.
  3. Créer une classe .important pour mettre en valeur certains paragraphes.
  4. Tester différentes polices disponibles sur le navigateur.

Rendu d’exemple

Titre Bleu

Paragraphe important en rouge.

Paragraphe normal en couleur sombre.

Astuces et bonnes pratiques

  • Préférer les couleurs contrastées pour l’accessibilité.
  • Limiter le nombre de polices pour garder l’uniformité.
  • Utiliser les couleurs hexadécimales ou rgba pour plus de contrôle.
