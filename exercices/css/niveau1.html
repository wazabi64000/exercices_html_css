CSS — Niveau 1 : Sélecteurs et syntaxe

Objectif

Apprendre la syntaxe de base du CSS et les sélecteurs simples pour appliquer des styles aux éléments HTML.

Exemple de code

/* Sélecteur par élément */
p {
  color: blue;
  font-size: 16px;
}

/* Sélecteur par classe */
.intro {
  font-weight: bold;
}

/* Sélecteur par id */
#titre {
  text-align: center;
}
  • Les règles CSS suivent le format sélecteur { propriété: valeur; }
  • Les sélecteurs peuvent cibler un élément, une classe (.nom) ou un id (#id).
  • La couleur et la taille de police sont des propriétés de base.

Exercices

  1. Créer un fichier style.css et lier-le à une page HTML existante.
  2. Appliquer une couleur différente à tous les p de la page.
  3. Créer une classe .important et l’appliquer à un paragraphe pour le mettre en gras.
  4. Créer un id #titre pour centrer un h1.
  5. Tester les changements dans le navigateur.

Rendu d’exemple

Titre Centré

Ce paragraphe est en gras et en couleur.

Paragraphe normal en couleur bleue.

Astuces et bonnes pratiques

  • Utiliser des classes plutôt que des ids pour la réutilisabilité.
  • Nommer les classes de manière explicite (.intro, .important).
  • Éviter de mettre du style directement dans les balises HTML (inline) sauf pour tests rapides.
  • Valider votre CSS avec un validateur comme W3C CSS Validator.
