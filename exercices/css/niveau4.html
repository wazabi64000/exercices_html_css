CSS — Niveau 4 : Flexbox et layout

Objectif

Apprendre à organiser des éléments en ligne ou en colonne avec Flexbox pour créer des mises en page réactives.

Exemple de code

.container {
  display: flex;
  flex-direction: row;
  justify-content: space-around;
  align-items: center;
}

.box {
  width: 100px;
  height: 100px;
  background-color: lightblue;
}
  • display: flex : active le modèle Flexbox.
  • flex-direction : ligne ou colonne.
  • justify-content : positionnement horizontal des enfants.
  • align-items : alignement vertical des enfants.

Exercices

  1. Créer une div.container contenant 3 div.box.
  2. Positionner les boîtes horizontalement avec espace égal.
  3. Changer la direction en colonne et observer l’effet.
  4. Tester différents alignements avec justify-content et align-items.

Astuces et bonnes pratiques

  • Flexbox est idéal pour les mises en page simples et réactives.
  • Combiner flex-wrap pour gérer les lignes multiples.
  • Tester sur différentes résolutions pour valider le comportement responsive.
