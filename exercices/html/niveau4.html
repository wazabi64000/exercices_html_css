HTML — Niveau 4 : Formulaires

Objectif

Apprendre à créer des formulaires interactifs avec différents types de champs et boutons.

Exemple de code

<form action="submit.php" method="post">
  <label for="nom">Nom:</label>
  <input type="text" id="nom" name="nom" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <label for="message">Message:</label>
  <textarea id="message" name="message"></textarea>

  <input type="submit" value="Envoyer">
</form>
  • Chaque champ doit avoir un label associé pour l’accessibilité.
  • Les types text, email, password sont les plus utilisés.
  • Le bouton submit envoie le formulaire.
  • L’attribut required rend le champ obligatoire.

Exercice

  1. Créer un formulaire de contact avec nom, email, message et bouton envoyer.
  2. Ajouter un select avec 3 options.
  3. Ajouter des checkbox ou radio pour des préférences.
  4. Tester l’envoi et la validation du formulaire.

Astuces et bonnes pratiques

  • Associer toujours label et id.
  • Utiliser les types adaptés pour chaque champ (email, number, etc.).
  • Tester la validation HTML5 avant JavaScript.
  • Maintenir un ordre logique des champs pour l’accessibilité.
