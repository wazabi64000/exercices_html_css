HTML — Niveau 2 : Sémantique

Objectif

Apprendre à utiliser les balises sémantiques pour structurer le contenu et améliorer l’accessibilité.

Exemple de structure

<header>
  <h1>Mon Blog</h1>
  <nav>...</nav>
</header>

<main>
  <article>
    <h2>Titre d'article</h2>
    <p>Contenu...</p>
    <time datetime="2025-10-07">7 octobre 2025</time>
  </article>
</main>

<footer>...</footer>

Explication

  • <header> : zone d’introduction, logo, navigation.
  • <main> : contenu principal unique par page.
  • <article> : unité autonome réutilisable.
  • <footer> : informations de bas de page.

Exercice

Créer article.html :

  • <header> avec logo et menu
  • <main> contenant un <article> complet (titre, date, auteur, paragraphes)
  • <aside> avec ressources connexes
  • <footer> avec mentions légales

Astuces

Un seul <h1> par page. Utiliser les niveaux d’en-têtes pour la hiérarchie. Préférer sémantique > <div>.
