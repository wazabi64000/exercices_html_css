HTML Bonus — Référentiel complet

Objectif

Fournir un inventaire complet des balises HTML, les obsolètes, les sémantiques, et bonnes pratiques.

Liste des balises HTML courantes

  • <a> : lien hypertexte
  • <p> : paragraphe
  • <h1>…<h6> : titres
  • <ul>/<ol>/<li> : listes
  • <div>/<span> : conteneurs génériques
  • <img> : image
  • <video>/<audio> : médias
  • <form>/<input>/<textarea>/<button> : formulaires
  • <table>/<tr>/<th>/<td> : tableaux
  • <header>/<footer>/<main>/<article>/<section>/<aside> : sémantique
  • <details>/<summary> : contenu interactif

Balises obsolètes

  • <font>, <center>, <b> pour mise en forme (préférez CSS)
  • <marquee> : animation texte, à éviter
  • <u> : underline purement décorative, utiliser CSS

Bonnes pratiques

  • Privilégier les balises sémantiques pour le SEO et l’accessibilité.
  • Ajouter systématiquement alt aux images.
  • Valider le HTML avec W3C.
  • Ne pas utiliser de balises obsolètes.
  • Structurer les titres hiérarchiquement (h1 → h2 → h3).
  • Maintenir une indentation propre et logique.
