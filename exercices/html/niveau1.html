HTML — Niveau 2 : Sémantique

Objectif

Apprendre les balises sémantiques pour améliorer la structure, le SEO et l’accessibilité de vos pages HTML.

Exemple de code

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>Exemple sémantique</title>
  </head>
  <body>
    <header>
      <h1>Mon site</h1>
    </header>
    <nav>
      <ul>
        <li><a href="#">Accueil</a></li>
        <li><a href="#">Contact</a></li>
      </ul>
    </nav>
    <main>
      <article>
        <h2>Article principal</h2>
        <p>Texte de l'article.</p>
      </article>
      <aside>
        <p>Informations complémentaires.</p>
      </aside>
    </main>
    <footer>
      <p>© 2025 Mon site</p>
    </footer>
  </body>
</html>
  • <header> : bloc d’introduction ou en-tête.
  • <nav> : menu de navigation principal.
  • <main> : contenu principal.
  • <article> : contenu indépendant (article, billet, post).
  • <aside> : contenu secondaire ou complémentaire.
  • <footer> : pied de page.

Exercice

  1. Créer une page avec un header, un nav, un main contenant deux article, et un aside.
  2. Ajouter un footer avec vos coordonnées ou copyright.
  3. Structurer correctement les titres h1 et h2.
  4. Tester la sémantique avec l’outil d’accessibilité de votre navigateur.

Astuces et bonnes pratiques

  • Utilisez toujours les balises sémantiques plutôt que des div génériques.
  • Maintenez la hiérarchie des titres (h1 → h2 → h3).
  • Testez l’accessibilité avec Lighthouse ou WAVE.
  • Commentez votre code pour clarifier la structure.
